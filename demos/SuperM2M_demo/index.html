Sound Demo of SuperM2M on CHiME-4 Corpus


This webpage provides a sound demo on the test set of the CHiME-4 corpus. For technical details, please refer to the following paper:

Z.-Q. Wang, "SuperM2M: Supervised and Mixture-to-Mixture Co-Learning for Speech Enhancement and Robust ASR", in submission, 2024.

This demo presents the enhancement results of 20 randomly-selected test signals.
1. 20 randomly-selected test signals in 1-channel input cases (click to jump to the section)
2. 20 randomly-selected test signals in 2-channel input cases (click to jump to the section)
3. 20 randomly-selected test signals in 6-channel input cases (click to jump to the section)
Firefox is recommended.
